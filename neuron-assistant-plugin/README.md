# Neuron Assistant Plugin (Optional)

This is a minimal Claude Code plugin skeleton to help generate Tentacle Neuron task payloads.

## Install (local)

1. Copy this folder into your project or user plugins directory.
2. Ensure the plugin directory contains `.claude-plugin/plugin.json`.
3. Restart Claude Code.

## Commands

- `/task-template` - Generate a minimal task JSON template.

## Notes

This plugin is optional and does not affect `tentacle-neuron` runtime. It is only a convenience for task authors.
